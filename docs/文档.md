# EGOEquipment（EGO 装备）

## 模块定位

管理**EGO 武器**和**EGO 护甲**的模组，包含近战/远程两大类数十种武器。依赖 GoldenBoughsLib 和 ResonatorCombatFramework。

## 项目结构

```
src/main/
├── java/architecture/ego_equipment/
│   ├── client/
│   │   ├── gui/hudlayers/              — HUD（GunChargeUpHudLayer 蓄力, RedEyesTachiChopFlavor 斩味）
│   │   ├── particle/                   — 粒子（DyeingMagicCircle + magicbullet/ + solemnlament/）
│   │   └── renderer/                   — 渲染器（entity/MagicBullet, item/MagicBulletMagicCircle, item/RedEyesTachi）
│   ├── common/
│   │   ├── entity/projectile/          — 弹射物（MagicBulletEntity, ParadiseLostSpikeweed）
│   │   └── item/
│   │       ├── armor/                  — EgoArmorItem
│   │       └── weapon/                 — 武器系统（详见下方武器分类）
│   ├── core/                           — EGOEquipmentConstants
│   │   └── registry/                   — 注册表（GuiLayer, ParticleProviders + client/EntityRenderers）
│   ├── datagen/                        — 数据生成（Datagen, ItemModel, Particle, Sound, DatapackBuiltin, i18n/ZhCn, BlockTag, ItemTag）
│   ├── events/                         — ClientEvents + client/PlayerRenderEvents
│   ├── init/                           — 初始化（ArmorItems, ArmorMaterials, CreativeModeTabs, MobEffects, ParticleTypes, ProjectileEntityTypes, SoundEvents, WeaponItems）
│   │   └── tag/                        — EquipmentItemTags
│   └── mixin/                          — MobEffectInstanceAccessorMixin（唯一 mixin）
│
└── kotlin/architecture/ego_equipment/
    └── core/                           — EGOEquipment.kt (@Mod), EGOEquipmentClient.kt
```

## 武器系统

### 近战武器（melee/）

| 子类型 | 目录         | 说明                                          |
|-----|------------|---------------------------------------------|
| 斧   | `axe/`     | AxeEgoWeaponItem, AxeEgoWeaponGeoItem       |
| 拳   | `fist/`    | FistEgoWeaponItem                           |
| 锤   | `hammer/`  | HammerEgoWeaponItem                         |
| 匕首  | `knife/`   | KnifeEgoWeaponItem                          |
| 钉头锤 | `mace/`    | MaceEgoWeaponItem                           |
| 矛   | `spear/`   | SpearEgoWeaponItem                          |
| 剑   | `swords/`  | SwordsEgoWeaponItem, SwordsEgoWeaponGeoItem |
| 特殊  | `special/` | 22 种（详见下方）                                  |

#### 近战特殊武器（22 种）

Amita, BlackSwan, Censored, CobaltScar, CrimsonScar, DaCapo, Discord, Ecstasy, Gaze, GoldRush, GrinderMk4, Justitia,
LifeForADaredevil, Mimicry, Moonlight, OurGalaxy, Pleasure, RedEyesTachi, Smile, TheSwordSharpenedWithTears, Twilight

### 远程武器（remote/）

| 子类型 | 目录          | 说明                                                                                                                                  |
|-----|-------------|-------------------------------------------------------------------------------------------------------------------------------------|
| 炮   | `cannon/`   | CannonEgoWeaponItem                                                                                                                 |
| 弩   | `crossbow/` | CrossbowEgoWeaponItem, CrossbowEgoWeaponGeoItem                                                                                     |
| 枪   | `gun/`      | GunEgoWeaponItem, PistolEgoWeaponItem, MagicBulletWeaponItem, SolemnLamentWeaponItem + pistol/PinkWeapon + rifle/RifleEgoWeaponItem |
| 特殊  | `special/`  | 9 种（Adoration, CherryBlossoms, EngulfingDream, FeatherOfHonor, InTheNameOfLoveAndHate, ParadiseLost, SoundOfAStar, Syrinx）          |

### 武器基类层次

```
EgoWeaponItem（顶层抽象）
├── MeleeEgoWeaponItem → MeleeEgoWeaponGeoItem
│   ├── AxeEgoWeaponItem → AxeEgoWeaponGeoItem
│   ├── SwordsEgoWeaponItem → SwordsEgoWeaponGeoItem
│   ├── FistEgoWeaponItem / HammerEgoWeaponItem / KnifeEgoWeaponItem / ...
│   └── special/*WeaponItem（22 种直接实现）
└── RemoteEgoWeaponItem → RemoteEgoWeaponGeoItem
    ├── CannonEgoWeaponItem / GunEgoWeaponItem / ...
    └── special/*WeaponItem（9 种直接实现）
```
